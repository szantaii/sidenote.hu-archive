Ha a legújabb Ubuntut használod (13.10), akkor nincs más tennivalód, mint feltenni ezt a két csomagot: exfat-fuse, exfat-utils.

Nyiss egy terminált, és ezt írd be:

sudo apt-get install exfat-fuse exfat-utils -y

.

Ha régebbi Ubuntud van, és az imént leírt megoldás nem működne, akkor ezeket a parancsokat hajtsd végre:

sudo apt-add-repository ppa:relan/exfat
sudo apt-get update
sudo apt-get install fuse-exfat exfat-utils
